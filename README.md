# oxidized-extend-models

Extended custom model for oxidized

## Included models
- `models/sonicnet.rb`: base SONiC-style model.

## Install
Copy the model files into your Oxidized model path (commonly `~/.config/oxidized/model/`) and reference the model name in your Oxidized source mapping.

## Notes
- These are intentionally minimal starter models.
- Update prompts/commands to match your device CLI and policy.
